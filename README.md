# Fish Shell Configuration

Aliases, function and environment for the Fish shell.

# Aliases / Abbreviations

## Commands

| cmd    | Action                            |
| ---    | ---                               |
| `..`   | `cd ..`                           |
| `...`  | `cd ../..`                        |
| `bak`  | `cp $file -> $file~`              |
| `cx`   | `chmod +x`                        |
| `fd`   | `fdfind`                          |
| `gits` | `git status`                      |
| `grep` | `grep --color`                    |
| `la`   | `ls --color=auto -lahN`           |
| `lar`  | `ls --color=auto -lahNtr`         |
| `ls`   | `ls --color=auto -N`              |
| `path` | `find exec in path or print path` |
| `psg`  | `ps -ef \| grep -i`               |
| `sw`   | `swap file names`                 |
| `tigs` | `tig status`                      |
| `tig`  | `tig --all`                       |
